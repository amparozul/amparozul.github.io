CSS Measuring Units

Absoulte vs. Relative

CSS length units fall in two categories: absolute and relavite

Absoulte units are fixed measurements thaat do not change based on other factors like viewport size or screen size. Examples include px (pixels),pt (points), in (inches) and cm(centimeters). These units are widely used for we designing.

Examples of Measuirng Units

1. px and pt

A pixel is oten representing a tiny dot on a screen. A point is a traditional print unit. They are rarelyy used for responsive design.

{ font-size :16px; /* fixed pixles size */ } h2 { font-size: 12pt; /* fixed point size */ }

2. em and rem (Relative Units)

Em:Relative to the font-size of the parent element. Multiple em values can actually become confusing.

Rem: Relative to the root element's font-size.

{
    font-size; 14px;
}
h1 {
    font-size 2rem; /* 24px, based on root */
}

 3. VW and VH (Viewport Units)


VW: Relative to the width of the viewport.



 VH: Relative to the height of the viewport.



 These units are useful for fullscreen layouts and scaling when windows are resized.
hero{ width: 100vw; /* width of screen */ height: 100vh; /* height of screen */ }
