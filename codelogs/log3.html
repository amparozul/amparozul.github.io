CSS Media Queries

What is a Media Query?

A Media Query is a technique allowing web developers to apply different styles to a webpage based on the characteristics of the device.

For example, media queries can be used to adapt designs to the form of a user's device.

Why are Media Queries Useful?

Media Queries are important because they allow anyone to create different layouts depending on the size of viewpoint.

Here are 4 commonly used media features:

1. Max-Width

This feature aims screens no wider than a certain number of pixels.

@media (max-width:768px)

2. Min-Height

Applies styles when the screen's height is a certain value.

@media (min-height:800px)

3. Orientation

Exposes if the device is in landscape or portrait mode.

@media (orietation: portrait)

4. Prefers-color-scheme

Exposes if the user prefers dark or light scheme.

@media (prefers-color-scheme: dark)

All these allow the site to correctly adapt to adapt to the user's theme preference.
